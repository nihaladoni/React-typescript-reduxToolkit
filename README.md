## ⚡ Getting started

1. Clone the repo

```sh
git clone https://github.com/nihaladoni/typescript-reduxToolkit.git
```

2. Change directory

```sh
 cd typescript-reduxToolkit
```

3. Install NPM packages

```sh
npm install
```

4. Run the app using command

```sh
npm start
```
